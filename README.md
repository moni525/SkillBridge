# SkillBridge

SkillBridge is an AI-powered platform that helps engineering students manage mentors, opportunities, and personalized learning paths in one place.

## Features
- AI learning path generator
- Mentor discovery
- Opportunity dashboard
- Project portfolio tracker

## Tech Stack
Frontend: React + Vite
Backend: Python (Flask)
AI: Gemini API
Database: SQLite

## Future Improvements
- Resume analyzer
- AI mentor chatbot
- Skill gap detection
